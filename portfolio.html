Bus Tracker
This project is a real-time bus tracker for routes between MIT and Harvard campuses. Stops are highlighted and buses are updated every second. This was built using JavaScript, html and CSS with asynchronous code for retrieving bus locations. This was an MIT project. Project was developed by Drew Quinn.
Pacman
This project was to demonstrate how to walk a Pacman back and forth across a screen within the variable screen width. Pacman's mouth opens and closes with each movement. This was built using JavaScript, html and CSS. This was an MIT project. Project was developed by Drew Quinn.
Eye Tracking
This project showed how to track the mouse curser, updating a set of eyes to follow. The eyes tracked the mouse wherever it moved across the screen, even outside the browser. This was built using JavaScript, html and CSS. This was an MIT project. Project was developed by Drew Quinn.
